# 0. Пара таблиц вместо введения

[![](https://cdn.discordapp.com/attachments/572801769516695564/572861715172294693/unknown.png)](https://cdn.discordapp.com/attachments/572801769516695564/572861715172294693/unknown.png)

Таблицы служат для вычисления расстояния до объекта по его известным размерам при разной крастности перископа. Например, по вертикали длинна корабля из справочника, по горизонтали количество рисок на перископе в которые он укладывается. Табличное значение требуется разделить на синус курсового угла!

Логарифмическая шкала справа служит для нахождения одного неизвестного из скорости (уз.), времени (мин.) и расстояния (км) по двум другим - строим прямую по двум известным величинам, она будет пересикать искомое значенние на оставшейся шкале.

# 1. Torpedo Data Computer (TDC)

[![](https://cdn.discordapp.com/attachments/572801769516695564/572866047577686055/unknown.png)](https://cdn.discordapp.com/attachments/572801769516695564/572866047577686055/unknown.png)

1)  Angle Tracking - выбираем, по какому перископу будем проводить запуск:
- A.P. - атакующий перископ;
- O.P. - зенитный перископ;
- UZO - устройство наведения на палубном мостике (бинокль там стоит такой);
- Off - задание угла по которому видим цель вручную на панели Heading;
2) Target Speed - скорость цели в узлах;
3) Torpedo Depth - глубина на которой пойдет торпеда, чем ближе к осадке цели, тем лучше;
4) Range - расстояние до цели в сотнях метров;
5) Length - -
6) Gyro Angle - угол, на который довернет торпеда;
7) Spread Angle - угол разброса торпед при залпе;
8) Time To Impact - время до цели;
9) Angle On Bow - курсовой уголь цели (КУЦ). Угол, под которым наблюдается наша лодка с цели: 0 - цель идет на нас, 180 - цель уходит от нас, "право 90" - мы справа от цели под прямым углом.
10) Impact Angle - угол удара торпеды, чем ближе к 90 тем лучше;
11) Лампа под IA - загорается, когда гарантирован промах (слишком далеко цель или она под сильным углом);
12) Heading - угол, под которым мы наблюдаем цель, если включен AT - не трогаем;
13) Torpedo Status - количество оставшихся торпед + готовность (лампа горит - торпеда готова к запуску). Сверху - носовой аппарат, снизу - кормовой;
14) Single Shot - выбор торпедного аппарата / залпа (Salvo);
15) Salvo Shot - выбор режима залпа, активен при установке Single Shot на Salvo;

Для успешного попадания по идее достаточно выставить СКОРОСТЬ, РАССТОЯНИЕ, КУРСОВОЙ УГОЛ, выбираем торпеду, ее глубину, перескоп наводки - идем к перископу и производим пуск. Дальше надеемся что цель не сменит курс.

# 2. Лодочные системы

## 2.1 Помпа

Включаем, чтобы откачать воду из обитаемых отсеков.

[![](https://cdn.discordapp.com/attachments/572801769516695564/572866945691549727/unknown.png)](https://cdn.discordapp.com/attachments/572801769516695564/572866945691549727/unknown.png)

## 2.2 Компрессор

Включаем НА ПОВЕРХНОСТИ, чтобы накачать воздух в баки (для последующей продувки балласта).

[![](https://cdn.discordapp.com/attachments/572801769516695564/572867053669580811/unknown.png)](https://cdn.discordapp.com/attachments/572801769516695564/572867053669580811/unknown.png)

## 2.3 Вентиля заполнения балласта

Открывает и закрывает доступ воде к балластным отсекам. Если не заполнить балластные отсеки водой - лодка не погрузится. Для носового и кормового отсеков вентиля заполнения свои.

[![](https://cdn.discordapp.com/attachments/572801769516695564/572867430251233300/unknown.png)](https://cdn.discordapp.com/attachments/572801769516695564/572867430251233300/unknown.png)

## 2.4 Вентиля балласта отрицательной плавучести

Верхний вентиль - заполнение водой, нижний - продувка воздухом. Левый прибор - уровень воды в балласте (оптимальный уровень - 0,1), правый - запас сжатого воздуха. Заполняется водой только для экстренного погружения!

[![](https://cdn.discordapp.com/attachments/572801769516695564/572867811064545290/unknown.png)](https://cdn.discordapp.com/attachments/572801769516695564/572867811064545290/unknown.png)

## 2.5 Вентиль продувки балласта

Продуваются только для экстренного всплытия или для увеличения скорости надводного хода. Центральный прибор - запас сжатого воздуха, боковые - количество воды в носовом и кормовом балластах.

[![](https://cdn.discordapp.com/attachments/572801769516695564/572868489862447124/unknown.png)](https://cdn.discordapp.com/attachments/572801769516695564/572868489862447124/unknown.png)

## 2.6 Руль глубины

- A - D - изменение угла рулей;
- Два верхних прибора - тахометры правой левой машины;
- Слева - мелководный глубинометр;
- Справа - глубоководный глубинометр;
- Справа внизу - угол рулей подводной лодки;
- По центру слева - глубинометр перископа;
- По центру справа - скорость погружения или всплытия;

[![](https://cdn.discordapp.com/attachments/572801769516695564/572869513083617281/unknown.png)](https://cdn.discordapp.com/attachments/572801769516695564/572869513083617281/unknown.png)

## 2.7 Машинный телеграф

AHEAD - вперед, ASTERN - назад. По центру кнопка тревоги. Чтобы управлять одновременно - зажимаем шифт.

[![](https://cdn.discordapp.com/attachments/572801769516695564/572869949756932096/unknown.png)](https://cdn.discordapp.com/attachments/572801769516695564/572869949756932096/unknown.png)

## 2.8 Рулевая машинка

- A - D - изменение угла рулей;
- Справа вверху -  счетчик количества энергии в аккумуляторах (килоамперчасметр);
- Слева - спидометр в узлах;
- По цетру - компас, показывает текущий курс;
- Справа - текущее положение рулей;

[![](https://cdn.discordapp.com/attachments/572801769516695564/572870522183090206/unknown.png)](https://cdn.discordapp.com/attachments/572801769516695564/572870522183090206/unknown.png)

## 2.9 Радио

Справа передатчик, слева приемник.

[![](https://cdn.discordapp.com/attachments/572801769516695564/572871006549442580/unknown.png)](https://cdn.discordapp.com/attachments/572801769516695564/572871006549442580/unknown.png)

Чтобы передать сообщение, берем телеграфный ключ и набираем текст, который до этого зашифровали Энигмой.

[![](https://cdn.discordapp.com/attachments/572801769516695564/572871233629323274/unknown.png)](https://cdn.discordapp.com/attachments/572801769516695564/572871233629323274/unknown.png)

## 2.10 Энигма

Пробелы не используются - они делают шифр уязвимым к частотному анализу.
Вверху выставляем ключ - три буквы ( по факту - две), жмем букву - Энигма высвечивает нам ее шифрбукву.

[![](https://cdn.discordapp.com/attachments/572801769516695564/572874962021384197/unknown.png)](https://cdn.discordapp.com/attachments/572801769516695564/572874962021384197/unknown.png)

## 2.11 Поисковик радиосигналов

Выставляем предполагаемое направление источника сигнала.

[![](https://cdn.discordapp.com/attachments/572801769516695564/572875677221650442/unknown.png)](https://cdn.discordapp.com/attachments/572801769516695564/572875677221650442/unknown.png)

Слева громкость, по центру выбор диапазона частот и частоты двумя крутилками.

[![](https://cdn.discordapp.com/attachments/572801769516695564/572875881115156511/unknown.png)](https://cdn.discordapp.com/attachments/572801769516695564/572875881115156511/unknown.png)

## 2.12 Гидрофон

- Вентиль внизу - изменение пеленга;
- Gain - усиление звука; 
- Внешний лимб - направление относительно курса лодки;
- Внутренний лимб - направление, относительно севера;
- Фильтры частот слева можно не трогать;

[![](https://cdn.discordapp.com/attachments/572801769516695564/572877083156545566/unknown.png)](https://cdn.discordapp.com/attachments/572801769516695564/572877083156545566/unknown.png)

## 2.13 Проигрыватель

Слева направо - комнаты куда играть, звук на лодке/звук снаружи, выбор и остановка музыки.
